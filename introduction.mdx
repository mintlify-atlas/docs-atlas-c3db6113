## Overview

CRM API is a comprehensive customer relationship management system designed for businesses to track and manage their sales pipeline effectively. Built with modern technologies, it provides a robust backend API powered by Spring Boot and an intuitive React-based frontend interface.

## Technology Stack

### Backend
- **Framework:** Spring Boot 3.0 (Java 17)
- **Database:** H2 Database (In-Memory)
- **Persistence:** Spring Data JPA / Hibernate
- **Validation:** Hibernate Validator
- **Testing:** TestNG

### Frontend
- **Library:** React.js (v18)
- **Routing:** React Router DOM (v6)
- **Validation:** Validator package
- **Testing:** Jest & React Testing Library

## Architecture

The application follows a clean, modular architecture:

- **Controllers:** RESTful endpoints for each resource (Users, Accounts, Contacts, Opportunities)
- **Services:** Business logic layer handling operations and validations
- **Repositories:** Data access layer using Spring Data JPA
- **Models:** JPA entities representing the domain model
